# Project-phase-1-Capstone

## Book Explorer Web App

This is a web app that lets you **browse books, search for them, and save your favorites**.  
It works on phones, tablets, and computers.  

---

## What You Can Do

- **See books** on the homepage  
- **Search books** by title  
- **Save your favorite books** so you can see them later  
- **Remove books** from your favorites if you want  
- Go between **Home, Favorites, and About** pages  

---

## How to Use It

1. Download or copy the project to your computer.  
2. Open the folder on your computer.  
3. Open a **live server** or any tool that can run HTML files (this makes the app work correctly in your browser).  
4. Open `index.html` to see the homepage.  
5. Click on books to add them to your favorites.  
6. Go to the Favorites page to see saved books.  

---

## Files You’ll See

- `index.html` → The homepage  
- `about.html` → About page  
- `favorite.html` → Favorites page  
- `home.js` → Controls homepage interactions  
- `favorite.js` → Controls favorites page  
- `fetchbook.js` → Gets book information from the internet
