# Project-phase-1-Capstone
# 📚 Book Explorer Web App – Frontend Phase 1 Capstone

Capstone-text-1 is a responsive and interactive web application that allows users to browse, search, and save their favorite books. This project demonstrates the use of HTML, Tailwind CSS, and JavaScript, including API integration, modular code, DOM manipulation, and localStorage for persistence.

# 🔹 Features

Responsive Homepage with hero section and book grid.

Search functionality to find books by title.

Favorites system: Add or remove books, stored in localStorage.

Multi-page layout: Home, Favorites, About.

Modern UI using Tailwind CSS for responsive, interactive, and visually appealing design.

API integration: Fetch book data from the Open Library API.

# 🛠️ Technologies Used

HTML5

Tailwind CSS

JavaScript (ES6 Modules)

Open Library API

LocalStorage

# 🚀 How to Run

## ⚠️ IMPORTANT: This app uses ES6 modules and requires a web server!

### Option 1: Using Node.js (Recommended)

1. Clone the repository:
```bash
git clone https://github.com/UWASEdjaria/Project-phase-1-Capstone.git
cd Project-phase-1-Capstone
```

2. Start the server:
```bash
node server.js
```

3. Open your browser and go to:
```
http://localhost:3000
```
### Option 2: Using VS Code Live Server Extension

1. Install "Live Server" extension in VS Code
2. Right-click on `lab1/index.html`
3. Select "Open with Live Server"

## 🎯 Explore the app:

- Browse books on the homepage
- Search for books using the search bar
- Add books to your favorites
- View your favorite books on the Favorites page
